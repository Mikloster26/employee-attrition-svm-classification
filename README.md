This project predicts employee attrition using Support Vector Machines (SVM), comparing multiple kernel functions to identify the best classification model.

🎯 Objectives
Encode categorical variables
Evaluate different SVM kernels
Analyze classification performance
🧠 Methodology
Applied pd.get_dummies()
Trained SVM models with:
Linear
Polynomial
RBF
Sigmoid
Evaluated confusion matrices and classification reports
📈 Metrics
Accuracy
Precision
Recall
ROC Curve
🔍 Insights
Kernel selection significantly impacts performance
RBF kernel typically provided better generalization
